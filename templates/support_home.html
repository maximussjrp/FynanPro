Para Colaboradores de Suporte

✅ Você pode:
  • Ver mensagens dos clientes
  • Responder no chat
  • Acessar respostas rápidas
  • Ver histórico de conversas
  • Marcar mensagens como lidas
❌ Você NÃO pode:
  • Ver dados financeiros dos usuários
  • Acessar transações
  • Ver relatórios financeiros
  • Modificar dados de usuários
  • Acessar configurações do sistema
Privacidade e Segurança: Este sistema foi projetado para proteger a privacidade dos usuários, permitindo apenas o acesso necessário para o suporte.
